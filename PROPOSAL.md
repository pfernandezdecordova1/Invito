# Invito - Mini Project #2 Proposal

## What I'm building:
An interactive party invitation app where users create custom party cards with event details, and the app generates a countdown timer and sharable invitation for guests.

## Who it's for / Why:
Party organizers need an easy way to create and share party details with friends. This app solves the problem of manually messaging everyone with event info and time, plus it builds excitement with a live countdown timer.

## Core features:
1. Create a party card (name, date, time, location, description)
2. Real-time countdown timer showing days/hours/minutes until party
3. Generate a sharable invitation link/card
4. View all created parties
5. Delete a party invitation
6. Copy invitation details to clipboard

## What I don't know yet:
- How to calculate time differences and countdown timers
- How to format date/time inputs for countdown
- How to generate sharable content (copy to clipboard)
- How to structure multiple party data
- Event handling for date/time inputs
