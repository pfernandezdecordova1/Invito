# INVITO - Mini Project #2 Proposal

## My Project Proposal

# What I'm building:

A luxury event curation platform called INVITO where users can create exclusive party invitations, explore the brand's story, and sign up as members to access private events worldwide.

# Who it's for? 

INVITO is for anyone who values meaningful connections — professionals seeking networking opportunities, creatives looking for collaborators, or people who simply want access to curated luxury gatherings. I chose this because I wanted to build something that feels like a real product with strong branding and multiple user interactions, not just a basic tool. The platform solves the problem of creating polished, shareable event invitations while also building an exclusive community around private events.

# Core features:
- Cinematic landing page with animated branding and a photo collage hero banner
- About Us section with the INVITO origin story and platform stats
- Create Party form (name, date, time, venue, description) that generates a personalized luxury invitation card you can print or copy
- Party card grid to manage all created events (view invitation, copy details, delete)
- Membership recruitment page with a benefits overview and sign-up form (name, email, phone, location)

# What I don't know yet:
- How to build a responsive CSS grid photo collage
- How to design a luxury invitation card purely with CSS (gradients, pseudo-elements, ornamental borders)
- How to manage multiple page views within a single HTML file using show/hide logic
- How to handle multiple forms and state arrays in vanilla JavaScript
- How to use the Clipboard API for sharing invitation text
